# Sesame Robot Micro — Bill of Materials (BOM)

> 本 BOM 参考原版 Sesame Robot 的电源/电子方案,针对 **Micro(2.1g 微型舵机四足)** 版本整理。
> 螺丝数量为**根据 CAD 孔位反推的估算值**(工程未建模螺丝本体),装配时以实物为准。
> 暂不含购买链接。

---

## 1. 控制 / 电子 (Electronics)

| # | 组件 | 型号 / 规格 | 数量 | 说明 |
|---|------|------------|------|------|
| E1 | 主控板 | **Seeed XIAO ESP32-S3** | 1 | 替代原"Metro Mini + WiFi 桥接"两块板,**单板集成主控 + WiFi/BLE**。3.3V 逻辑,11× GPIO,WiFi+BLE。 |
| E2 | OLED 显示屏 | **ER-OLEDM0.49**(0.49" **64×32**,SSD1306,I²C) | 1 | 显示表情;I²C 地址 `0x3C`,3.3V 供电。 |

> 排针/载板已移至 **第 6 节 耗材**(现有材料剪裁即可,无需采购)。

> ⚠️ **引脚预算**:8 舵机 + I²C OLED = 10/11 GPIO,WiFi 走片内不占脚。**离散脚已用满**(I²C 总线仍可挂更多传感器)。
> ⚠️ **PWM**:S3 的 LEDC 仅 8 路,8 舵机正好占满;固件用 `ESP32Servo`(非 `Servo.h`)。
> 🔌 视觉升级可选 **XIAO ESP32-S3 Sense**(同尺寸、同 MCU,带摄像头/麦克/SD);但 8 舵机 + 摄像头会撞 LEDC,建议视觉单独一颗芯片。

## 3. 电源系统 (Power)

> 方案:**1S 直供 / 无 DC-DC**(参考 [flix](https://github.com/okalachev/flix) 1S 直供有刷电机的成熟做法)。全机无器件强制要 5V(见 §1 与 §6 各器件电压:XIAO/OLED 走 3.3V,舵机 3.7–6.0V)。

| # | 组件 | 型号 / 规格 | 数量 | 说明 |
|---|------|------------|------|------|
| P1 | 电池 | **1S LiPo 3.7V** ·主选 **102050**(~600–1000mAh)/留极耳余量选 **102040·802040**(~500–800mAh) | 1 | **直供全场**。合机身底部电池仓 ≈ **20×10×50mm**(CAD 建模电池块同此尺寸)。满电 4.2V / 空 3.0V,落在舵机 3.7–6.0V 内。⚠️ 极耳+导线超本体 5–15mm,仓仅 50mm → 纯 102050 偏顶,**优先本体 40mm 款**。**必带保护板(PCM)或固件 ADC 低压切断(防过放)**。出厂多为 JST-PH 2.0;XIAO 是 BAT 焊盘,直焊或加转接线。 |
| P2 | 电源开关 | **SPDT Mini Slide Switch** | 1 | 串在电池正极与负载之间(CAD 已建模该开关位)。 |
| P3 | 滤波电容 | 电解电容 470–1000µF / ≥10V | 1 | 接电池 V+ 轨,吸收舵机/WiFi 电流尖峰,稳压。 |
| P4 | 升压稳压(**可选**) | 1S→5V 升压:**MT3608**(调压 ~36×17mm)/ **TPS61023**(小 ~18×16mm)/ **PowerBoost 500C**(升压+1S充电二合一 ~36×22mm) | 0–1 | **默认不用**(参考 flix 的"可选稳压")。仅当 WiFi 发射偶发欠压/复位才加,调到 5.0V 给舵机轨。⚠️ 升压抬高电池电流(5V2A≈从电池取 3A/~4C),**先试加大 P3 电容**再考虑升压。 |

**供电拓扑(1S 直供):**
```
1S LiPo(3.0–4.2V)
   └ 开关(P2) ─┬─→ XIAO ESP32-S3  BAT 焊盘   (直供 + USB-C 板载充电)
               │        └─ XIAO 3V3 脚 → OLED VCC
               └─→ 8× 舵机 V+(3.7–4.2V 直供)
   V+ 轨并 P3 大电容;全部共地(舵机棕线 / XIAO GND / OLED GND)
   无 Buck / 无 Boost / 无外置充电板;升压 P4 仅作可选稳压
```

## 4. 紧固件 (Fasteners)

> 数量为按 CAD 孔径反推的**估算**,以实物装配为准。

> 参考原版 Sesame Robot BOM 的"两种螺丝"方案;数量为按 CAD 孔径反推的**估算**,以实物装配为准。**只需采购 2 种螺丝。**

| # | 组件 | 规格 | 数量(估) | 用途(对应孔) |
|---|------|------|----------|----------------|
| **F1** | **M2 自攻螺丝(盘头,塑料自攻)** | **M2×5** 为主(备少量 **M2×8/×10** 穿长孔) | ~40 | **所有塑料结构**:腿/髋支架↔舵机座、internal-frame、OLED 固定、外壳。直接拧打印件底孔(Ø1.5–1.7)。**实测咬合深度 2.5–3mm → M2×5 正合**;极少数穿多段腿件的长通孔(深 6.5–12.5mm)用 M2×8/×10。**(=原方案主力自攻)** |
| **F2** | **M3 机制螺丝(内六角杯头 SHCS)** | **M3×10**(载板类)& **M3×12**(髋部深沉孔) | ~12 | **电路载板 + 4× 髋部支架**的合装(过孔 Ø3.2 + 杯头沉孔 Ø5.5)。**实测**:载板夹层 6mm(沉孔 2.5mm)→ M3×10;髋部夹层 ≈8.5mm(沉孔 6mm)→ M3×12。**(=原方案第二种机制螺丝的位置)** |
| — | 舵盘螺丝 | 舵机 horn 小螺丝(~M1.4) | — | **DM-S0020 自带,无需单买**(对应 Ø0.6–0.8 孔)。 |

> 配件(非螺丝种类):F2 的 Ø3.2+Ø5.5 是过孔+沉孔,对面需 **M3 螺母**或 **M3 热熔铜螺母**(经常拆建议热熔);用电烙铁压入。
> 原则:**结构/外壳 → M2 自攻;电路载板 + 髋部支架合装 → M3 机制;舵盘 → 舵机自带螺丝。**
